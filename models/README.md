# /models — VRM Avatar Models

Place `.vrm` files here. NightClaw loads them automatically.

**Getting VRM models:**
- [VRoid Studio](https://vroid.com/en/studio) — Free. Create custom anime avatars.
- [VRoid Hub](https://hub.vroid.com/) — Browse 100K+ community models.
- [Booth.pm](https://booth.pm/) — Japanese marketplace with paid and free VRM models.
- Commission from a 3D artist on Fiverr, Skeb, or Twitter.

**Model files are NOT included in this repo.** Each user provides their own.

See [docs/avatar-creation-guide.md](../docs/avatar-creation-guide.md) for details.
